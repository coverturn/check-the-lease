# Round 8.9 — The single move that transforms the visual: hard-offset shadows

This round is tight and surgical. One move applied everywhere: **hard-offset solid-colour shadows on every card, button, and badge**. Not blurry CSS shadows. Solid colour offsets, sliding forward on hover.

This is the move that comic books, Linear, Wispr Flow, and Manuel Neto's illustrations all use. It's THE thing missing.

---

# Round 8.9 — Hard-offset shadows everywhere

## What's gone wrong

After Rounds 8.7 and 8.8, the cards and CTAs still look like generic CSS rectangles with borders. The problem isn't the border weight or the colour — it's that they have **no visual weight or "drawn" quality**.

The fix is one specific move: **every interactive element gets a hard-offset solid-colour shadow**. No blur. No softness. Just a coloured rectangle offset behind the element by 4-6px.

## What to apply

### Cards — hard offset shadow

Every card on the site:

```css
.card-mn {
  background: #FBF8F1;
  border: 2.5px solid #171717;
  border-radius: 16px;
  padding: 24px;
  position: relative;
  z-index: 1;
  /* Hard offset shadow */
  box-shadow: 6px 6px 0 0 #171717;
  transition: transform 0.15s ease, box-shadow 0.15s ease;
}

.card-mn:hover {
  transform: translate(3px, 3px);
  box-shadow: 3px 3px 0 0 #171717;
}

.card-mn:active {
  transform: translate(6px, 6px);
  box-shadow: 0 0 0 0 #171717;
}
```

Featured (dark) cards get a **coloured shadow** instead of black:

```css
.card-mn-dark {
  background: #1E3A5F;
  color: #FBF8F1;
  border: 2.5px solid #171717;
  border-radius: 16px;
  padding: 24px;
  position: relative;
  /* Hard offset shadow in clay or sage to add colour */
  box-shadow: 6px 6px 0 0 #C97A4A;  /* or #5A8B7A for sage variant */
}
```

Variations to spread across cards (so they don't all use black shadow — adds vibrancy):

| Card type | Shadow colour |
|---|---|
| Standard cream cards | `#171717` (black, primary) |
| Featured/dark ink-blue cards | `#C97A4A` (clay) or `#5A8B7A` (sage) |
| Severity high (wine bg) | `#171717` (black) |
| Severity medium (clay bg) | `#171717` (black) |
| Resources cards on dark | `#5A8B7A` (sage) |
| "Why we built this" mission card | `#F5C547` (sunshine) |

The variety of shadow colours adds rhythm to the page. Black for the calm sections, brand-colour for moments that should pop.

## Why this works

A hard-offset shadow:
1. **Gives every element visual weight** — they don't float, they sit
2. **Makes hover physically meaningful** — the element slides forward and lands on top of where its shadow was
3. **Reads as "drawn" not "rendered"** — comic books, screen-printed posters, illustrated UI all use this
4. **Adds colour rhythm** — varying the shadow colours (black / clay / sage / sunshine) gives the page texture without overwhelming
5. **Makes the page feel like one connected illustrated world** — every interactive element has the same underlying geometry

This is the **single most missing move** in the current build.

## What this round REMOVES

If anywhere is still using:
- `box-shadow` with `blur` (e.g., `box-shadow: 0 4px 20px rgba(...)`) — REMOVE, replace with hard offset above
- `backdrop-filter` glassmorphism — REMOVE
- Drop shadows on illustrations — REMOVE (illustrations don't get shadows)

The ONLY box-shadow allowed is the hard solid-colour offset.
