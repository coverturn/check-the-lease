# Round 8.8 — Full Manuel Neto design language

This round applies Manuel Neto's design DNA across **everything**: icons, animations, graphic elements, colour blocking, and the colour palette itself. The goal is to take Wispr Flow's editorial calm and inject Manuel Neto's playful rubberhose energy throughout.

Inspiration: **manuelneto on Dribbble** — tagged "2d animation · character design · flat illustration · motion · rubberhose · rig · vector." Brazilian motion designer known for bold black outlines, flat fills, springy elastic motion, friendly characters, and floating decorative shapes.

# Round 8.8 — Manuel Neto-influenced design system

## Manuel Neto's signature, distilled

| Principle | What it means | How we apply it |
|---|---|---|
| **Bold black outlines** | 2-3px ink-black strokes on every illustrated element | Every card, button, badge, icon gets the same outline weight |
| **Flat fills, no gradients** | Solid colour blocks, never washes | All UI elements use flat fills |
| **Vibrant accents** | Brighter pops of colour against neutral grounds | Add 3 brighter accent tones to the existing palette |
| **Rubberhose motion** | Bouncy, elastic, springy with squash and stretch | Spring easing on all transitions, overshoot on appearance |
| **Floating decorative shapes** | Small geometric or character elements scattered around the main subject | Every section gets 2-4 floating shapes |
| **Hand-drawn quality** | Slight wobble, slight imperfection | Subtle rotation on cards, dashed lines for "drawn" feel |
| **Character-led** | Friendly, playful tone — illustrations have personality | All icons get rounded, friendly shapes |
| **Bold colour blocking** | Sections alternate between bold colour blocks, not subtle variations | Section backgrounds alternate colours decisively |

**Where to use the new accents:**

- **Peach `#F4A480`** — decorative shapes around hero, icon backgrounds in the "What you'll see" cards.
- **Lilac `#D4B8E5`** — floating decorative shapes in mid-page sections, subtle background variations.

**Severity colours stay muted** (clay/wine/sage). Never use the brights for severity — they'd undermine the calm legal tone.

## Part 2 — Colour blocking pattern for sections

Currently sections alternate bone → bone-dark. Make this **bolder**, not subtler. Manuel Neto uses decisive colour-block transitions, not subtle gradients.

New section-by-section background pattern:

| Section | Background | Reason |
|---|---|---|
| Hero | `#FBF8F1` (bone) | Calm entry |
| Marquee divider | transparent | Threading |
| The Problem | `#1E3A5F` (ink-blue, dark) | Grave, stops you scrolling |
| What is Check the Lease | `#FBF8F1` (bone) | Reset to calm |
| What you get | `#F2EDE2` (bone-dark) | Subtle cream variation |
| How it works | `#FBF8F1` (bone) | Reset |
| What you'll see | `#1E3A5F` (ink-blue) with cream cards | Bold contrast block |
| State preview | `#FBF8F1` (bone) | Reset |
| Who it's for | `#F2EDE2` (bone-dark) | Subtle variation |
| Without us / With us | `#FBF8F1` (bone) | Reset |
| Why we built this | `#1E3A5F` (ink-blue) | Mission moment |
| FAQ | `#FBF8F1` (bone) | Reset |
| Final CTA | `#1E3A5F` (ink-blue) | Closer |

The pattern: **cream → DARK BLOCK → cream → cream-dark → cream → DARK BLOCK → ...** Every other section is a bold colour block. The dark blocks stop you scrolling and force attention.

Each section transition is a sharp edge — no gradients, no fades. Bone meets ink-blue with a hard line.

## Part 5 — Floating decorative shape system

Every major section gets 2-4 floating decorative shapes in the corners or empty space. This is what makes Manuel Neto's work feel alive.

### Shape vocabulary (choose 4-6 base shapes)


### Where to place them

Per section, scatter 2-4 of these absolute-positioned around the corners and edges. Each gets `drift-float` animation with a different duration (6s, 8s, 10s, 12s) so they move asynchronously. Different rotation offsets so they don't all wave in unison.

Example placement around hero:
- Top-left: green circle (2-3% from edge)
- Top-right: clay triangle
- Mid-right: yellow star
- Bottom-left: lilac cross
- Mid-left: ink-blue rotated square

Vary the placement per section — don't repeat the same layout. Keep the shapes small (20-40px) so they're decoration, not focus.

## Part 6 — Animation vocabulary (extends Round 8.7)

All the keyframes from Round 8.7 still apply. Plus add:

```css
@keyframes wobble-idle {
  0%, 100% { transform: rotate(-1deg); }
  50% { transform: rotate(1deg); }
}

@keyframes blob-bob {
  0%, 100% { transform: translateY(0) scale(1); }
  50% { transform: translateY(-3px) scale(1.05); }
}

@keyframes card-tilt-on-hover {
  /* Already covered by transform on :hover */
}

@keyframes star-twinkle {
  0%, 100% { transform: scale(1) rotate(0); opacity: 1; }
  50% { transform: scale(1.2) rotate(15deg); opacity: 0.7; }
}
```

Apply:
- **wobble-idle** to featured cards (subtle rotation back and forth, 4s)
- **blob-bob** to the floating shapes that aren't already drifting (alternates with drift-float)
- **star-twinkle** to all sunshine yellow stars (3s ease-in-out infinite)
