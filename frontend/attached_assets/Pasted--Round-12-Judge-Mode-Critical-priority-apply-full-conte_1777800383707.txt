## Fix 1 — Flip the hero primary CTA to "See a real result →"

This is the single highest-leverage change in the round. Judges will not upload a real lease. They WILL click a one-click demo. Right now the demo CTA is secondary. Promote it.

**Find the current hero CTA layout:**
- Primary button: "Read my lease →"
- Secondary link: "Try a sample lease →"

**Replace with:**
- Primary button: "See a real result →" → links to `/results/demo`
- Secondary link: "Or read your own lease →" → links to `/upload`

**Why:** the strongest proof on the entire site is `/results/demo`. Putting it one click from the hero converts 100% of curious judges. The button copy "See a real result" frames it as the demo experience without using the word "demo" (which can read as "fake").

**Same flip on `/upload` page:** the link "Try the sample California lease →" should become a primary button styled identically to "Read my lease →" — placed RIGHT NEXT to the upload dropzone, not above it as a secondary link.

## Fix 11 — Lighthouse audit + fix anything below 90

Run Lighthouse on the deployed homepage. Target: 90+ on all four scores (Performance, Accessibility, Best Practices, SEO).

Most likely failure points and the surgical fixes:

**Performance below 90:**
- The state map SVG is heavy. Lazy-load it with `loading="lazy"` if it's an `<img>` or wrap the component in a `IntersectionObserver` so it only renders when scrolled to.
- The Anthropic chatbot script may be loading eagerly. Defer it until the user opens the bubble.
- The 6 hand-drawn 120×120 illustrations should be inlined (already are) but check for layout shift — they may need explicit `width/height` to prevent CLS.

**Accessibility below 95:**
- Verify `lang="en"` on `<html>`.
- Verify every interactive element has a visible focus ring.
- The CTA button contrast (sage on cream) — confirm it's still passing AA.
- Floating decorative SVGs all have `aria-hidden="true"`.
- Marquee text scroll has `prefers-reduced-motion` support.

**Best Practices:**
- All images use HTTPS (auto on Replit deploy).
- No console errors (check `read_console_messages`).
- No deprecated APIs.

**SEO:**
- Meta description present (it is).
- `<title>` is unique and descriptive (it is).
- Robots meta tag set to `index, follow` (NOT `noindex` from a forgotten dev flag).
- Sitemap.xml exists at root.
