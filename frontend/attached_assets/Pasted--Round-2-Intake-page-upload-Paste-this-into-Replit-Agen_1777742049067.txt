#### Card 1 — Upload your lease

Eyebrow: `01 — UPLOAD`
Heading: "Drop your lease."
Body: "PDF only. Up to 25MB. Stays in your browser — never stored on our servers."

The input: a drag-drop zone built with native HTML5 drag-drop events (DON'T install react-dropzone).

States:
- **Empty:** dashed 2px border in soft-opacity, padding 48px, centred content. Show Lucide `<FileUp size={28} className="text-sage" />` icon centred above text. Below icon: "Drop your PDF here, or click to browse." (Inter 14px ink). Below that: "PDF · max 25MB" (Inter 11px ink-muted). Whole zone is clickable to open file picker (accept=".pdf"). Hover: border switches to solid 2px medium opacity.
- **File selected:** solid 1.5px medium-opacity border, padding 16px, flex row. Left side: Lucide `<FileText />` icon + filename (Inter 14px ink, weight 500) + filesize (Inter 11px ink-muted) below. Right side: ghost button with Lucide `<X />` 16px to clear the file.

## State management

Use React `useState` only — don't add Zustand or Redux. Form shape:

```ts
type IntakeState = {
  file: File | null;
  state: string | null;
  stage: 'before' | 'already';
  perspective: 'renter' | 'landlord' | 'both';
  isParent: boolean;
  receivesHousingAid: boolean;
  isStudent: boolean;
  reviewingForSomeoneElse: boolean;
};
```

Initial values: `{ file: null, state: null, stage: 'before', perspective: 'renter', isParent: false, receivesHousingAid: false, isStudent: false, reviewingForSomeoneElse: false }`.

On submit (only when file is set AND state is set): navigate to `/results/demo` using `useRouter().push('/results/demo')`. The /results page doesn't exist yet — that's expected, we'll build it in Round 3.

Persist the intake state to `sessionStorage` under key `ctl-intake` so Round 3 can read it back on the results page.
